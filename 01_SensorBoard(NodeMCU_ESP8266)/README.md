# 01_SensorBoard(NodeMCU_ESP8266) 
### : NodeMCU와 여러 센서들을 연결해 실시간으로 웹과 앱으로 센서들의 값을 확인 할 수 있는 가정용 센서보드

# 기능
### 1) 시간, 온도, 습도 LCD로 표시
### 2) 유해가스 누출시 부저 및 적색LED On, LED 경고 표시
### 3) 밝기 따라 무드등(녹색)LED On
### 4) Iot Platform 인 ThingSpeack 통해 실시간으로 웹, 앱으로 센서 값 확인
### 5)
