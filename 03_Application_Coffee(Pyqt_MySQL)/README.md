## 03_ApplicationCoffee(Pyqt5, MySQL)
#### : Pyqt와 DB를 연동해서 카페에서 메뉴 추가, 가격 수정, 삭제, 판매세부내역 확인 할 수 있는 애플리케이션

## 기능
### 1) 제품 검색, 추가, 삭제, 수정
### 2) 가격 검색, 수정, 제품 추가
### 3) 판매세부내역 판매가격으로 정렬(오름차순, 내림차순)
### 4) 판매세부내역 마진가격으로 정렬(오름차순, 내림차순)
