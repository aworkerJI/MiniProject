# 02_VideoProcessor(RaspberryPi3) 
### : 라즈베리3와 Usb카메라로 QtCreator, OpenCv 이용해 다양한 영상처리와, 캡쳐, 녹화등이 가능한 애플리케이션

## 기능
### 1) 버튼 클릭에 따라 이동, 회전, 크기변화
### 2) RGB 채널분리, 병합
### 3) 영상을 히스토그램 처리
### 4) 차와, 얼굴 인식 및 표시
### 5) 조도센서의 값에 따라 영상 밝기 조절 
### 6) 캡쳐 후 자동저장
### 7) REC 클릭시 영상 녹화 및 LED(적색) On, Stop 클릭시 녹화 종료 및 LED(적색) Off
